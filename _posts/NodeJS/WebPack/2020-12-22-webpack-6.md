## 주요속성

* entry
* output
* loader
* plugin

```js
var path = require('path');

module.exports = {
	mode: 'production',
	entry: './js/app.js',
	output: {
			path: path.resolve(__dirname, 'build'),
			filename: 'main.bundle.js'
	},
	module: {
		rules: [{
			test: /\.m?js$/,
			exclude: /(node_modules|bower_components)/,
			use: {
				loader: 'babel-loader',
				options: {
					presets: ['@babel/preset-env']
				}
			}
		}]
	},
	stats: {
			colors: true
	},
	devtool: 'source-map'
};
```

---

## entry

웹팩에서 웹 자원을 변환하기 위해 필요한 최초 진입점의 경로

```js
module.exports = {
	entry: './js/app.js',
```

---

## output

웹팩을 돌리고 난 결과물의 경로

```js
module.exports = {
	output: {
			path: path.resolve(__dirname, 'build'),
			filename: 'main.bundle.js'
            // [name], [hash], [chunkhash]와 같은 해시 값을 붙일 수 있다
	},
```

---

## loader

웹팩이 웹 어플리케이션을 해석할 때 자바스크립트 파일이 아닌 웹 자원(HTML, CSS, image, font)들을 변환할 수 있도록 도와주는 속성

```js
module.exports = {
	module: {
		rules: [{
			test: /\.m?js$/,
			exclude: /(node_modules|bower_components)/,
			use: {
				loader: 'babel-loader',
				options: {
					presets: ['@babel/preset-env']
				}
			}
		}]
	},
```
